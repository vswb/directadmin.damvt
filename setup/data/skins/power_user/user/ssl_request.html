SSL Certificate Request
|NOTICE|
Here is your SSL Certificate Request. You will send this text to a Certificate Authority and they will give you a Signed Certificate. To use it, paste the Signed Certificate on the next line after
"-----END RSA PRIVATE KEY-----" on the previous page and click "Save".
(IE only)
|*if USERTYPE="admin"| Be sure to copy and backup the "RSA PRIVATE KEY" along with the the request. You will need it when installing the certificate. Because you're an admin, you will be saving to the "shared server certificate" and the key is not saved anywhere until you paste it with the certificate. |*endif| |HTM_USER_BOTTOM|
